Decision Support Tool - Excel VBA

The project was created with four other team members. This program helps you to plan and execute a project. You are able to see each team member's schedule, search questions, track the progress of the project, and make a cause-and-effect diagram (ishikawa or fishbone).

Budgeting Calendar - Processing

The Budgeting Calendar allows a user to select a range of dates for their budget. Then they can enter specific dates that they expect revenues and expenses. Once everything is entered, the user can view it on a calendar with a summary on the side. There is also a feature to save a picture of the calendar.
